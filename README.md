
# 📺 YTube – Video Streaming Mobile App

YTube is a modern video browsing mobile application inspired by YouTube, built using Flutter.  
It provides a clean user interface, smooth navigation, and dynamic video content display for a simple and distraction-free experience.

---

## 🚀 Overview

YTube is a lightweight YouTube-style application designed to demonstrate video browsing, UI design, and mobile application development using Flutter.

This project was developed for **personal use and learning purposes**, focusing on implementing real-world features such as API integration and modern UI design.

The application integrates the **YouTube Data API** to fetch and display video content dynamically.

---

## 👨‍💻 My Role

- ✅ Flutter Developer

- ✅ Integrated YouTube Data API for fetching video data

---

## ✨ Features

- 🎬 Browse video content using YouTube API  
- 🔍 Display video details (title, thumbnails, etc.)  
- 🚫 Ad-free browsing experience  
- 📱 Clean and modern UI design  
- 🔄 Smooth navigation between screens  
- ⚡ Optimized mobile performance  

---

## 🛠 Technologies Used

- **Framework:** Flutter  
- **Language:** Dart  
- **API:** YouTube Data API v3  
- **Tools:** Android Studio, VS Code
